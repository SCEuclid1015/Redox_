About Project

The Humanoid Robot project aims to create an interactive humanoid robot that can perform basic tasks and respond to voice commands. Using the ESP32-CAM microcontroller, this robot leverages advanced technologies such as computer vision and machine learning to enhance its functionality. Equipped with a built-in camera, the ESP32-CAM allows the robot to recognize objects and navigate its environment autonomously.

  • Autonomous navigation within its environment.

Why We Decided to Do This Project

We chose to undertake this project to explore the intersection of robotics and artificial intelligence. The growing demand for automated solutions in various fields inspired us to create a humanoid robot capable of performing tasks that can assist in daily life. Our goal is to innovate and understand the challenges in building intelligent machines while developing our skills in programming and hardware integration.
